## Project Description
This project is a web-based Scholarship Eligibility Management System developed using HTML, CSS, JavaScript, and JSON Server.

The system allows students to:
- Register an account
- View scholarship eligibility criteria
- Upload required documents
- Submit scholarship applications
- Receive application confirmation

## Technologies Used
- HTML5
- CSS3
- JavaScript
- JSON Server
- GitHub
